How to Install
  1. Click the Download button above to get the Nebula Browser installer (EXE).
  2. Double-click the downloaded file to start the setup.
  3. If you see a Windows SmartScreen warning, click More info → Run anyway.
    (This is normal for new apps. Nebula Browser is safe!)
  4. Follow the on-screen instructions to complete installation.
  5. Launch Nebula Browser from your desktop or Start menu and enjoy!
Note: Windows may show a SmartScreen warning for new apps. Nebula Browser is safe and does not contain any malware. Click More info → Run anyway to proceed.
